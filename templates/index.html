Welcome to Hello Flask


This is some boiler-plate code to launch the server behind this page: 
from flask import Flask

app = Flask(__name__)

@app.route('/')
def main():
    return render_template('index.html')
    
if __name__ == '__main__':
    app.run(threaded=True, debug=True)
The parts that matter here are:

@app.route('/')
What follows are the instructions for what to return when the user browses to /

return render_template('index.html')
Render the template index.html

app.run()
Start the server. The default is to run on the loopback address 127.0.0.1, on port 5000.
